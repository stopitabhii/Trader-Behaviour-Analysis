# Trader Behaviour Analysis using Bitcoin Fear & Greed Index

## Objective

The goal of this project is to explore the relationship between Bitcoin market sentiment and trader performance using historical trading data from Hyperliquid and the Bitcoin Fear & Greed Index.

---

## Datasets

### 1. Historical Trader Data
Contains:
- Coin
- Closed PnL
- Trade Size (USD)
- Direction
- Fee
- Timestamp

### 2. Bitcoin Fear & Greed Index
Contains:
- Date
- Fear & Greed Value
- Market Sentiment Classification

---

## Methodology

- Data cleaning
- Timestamp preprocessing
- Date standardization
- Dataset merging
- Exploratory Data Analysis (EDA)
- Data visualization
- Market sentiment analysis

---

## Key Insights

- Fear accounted for the highest number of trades in the dataset.
- Extreme Greed showed the highest average Closed PnL.
- Median Closed PnL remained zero across all market sentiment categories, indicating a highly skewed profit distribution.
- Trading volume and average trade size varied across different market sentiments.
- Market sentiment appears to influence overall trader profitability.
